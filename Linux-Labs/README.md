Linux Labs

This folder contains my Linux practice labs, system administration exercises, and command-line operations completed as part of my cybersecurity and IT training.

Contents

- Basic Linux Commands 
  Navigation, file creation, directory management, and text viewing commands.

- User & Permission Management
  Creating users, modifying groups, changing file permissions, and ownership exercises.

- Process & System Monitoring  
  Using tools like top, htop, ps, df, du, and system logs.

- Networking Commands 
  Testing connectivity and troubleshooting with ping, netstat, ifconfig/ip, traceroute, etc.

- Bash Shell Exercises  
  Looping, conditional statements, and script creation.

- Log Inspection Labs  
  Working with /var/log, searching logs using grep, awk, and basic incident findings.

Skills Demonstrated

- Linux command-line proficiency  
- File system navigation and management  
- Shell scripting fundamentals  
- User, group, and permission control  
- System and process monitoring  
- Log file analysis  
- Troubleshooting and network diagnostics

 Purpose

These labs demonstrate hands-on Linux skills essential for cybersecurity roles, including system hardening, troubleshooting, and analysis.
